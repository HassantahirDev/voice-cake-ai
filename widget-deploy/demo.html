üöÄ Quick Integration

Add this single line to your website's <head> section:

<script src="https://your-cdn-domain.com/voicecake-widget.js" data-agent-id="49" data-position="bottom-right" data-theme="light"></script>

That's it! The widget will automatically appear on your website.

üîß Advanced Integration

For more control, you can initialize the widget programmatically:

// Load the widget script const script = document.createElement('script'); script.src = 'https://your-cdn-domain.com/voicecake-widget.js'; document.head.appendChild(script); // Initialize when ready script.onload = function() { window.VoiceCakeWidget.init({ agentId: '49', position: 'bottom-right', theme: 'light', size: 'medium' }); };
